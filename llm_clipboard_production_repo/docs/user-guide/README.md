# Privacify User Install Guide

This folder contains the end-user installation guide in Word and PDF formats.

- [Word guide](Privacify_User_Install_Guide.docx)
- [PDF guide](Privacify_User_Install_Guide.pdf)
- `screenshots/` contains the current local manager UI snapshots referenced by the guide.

The guide is written for users installing and using Privacify. It does not include internal release, signing, or social-post guidance.
